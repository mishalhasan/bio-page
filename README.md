# Mishal Hasan — Personal Portfolio Website

This repository contains my **personal portfolio website**, built as part of a bootcamp project. The site highlights my **skills, projects, and professional background**, demonstrating my ability to build structured, semantic, and responsive web pages.

---

## About the Project

The website features:

- **About Me**: Professional bio highlighting full-stack development skills and personal interests
- **Skills & Experience**: Technical skills, professional development, and industry experience
- **Projects**: A showcase of my projects, including **Eira — Personal Health Companion**
- **Contact**: A functional contact form and links to email, LinkedIn, and GitHub

The project was built using **HTML5** with semantic structure, focusing on accessibility, proper heading hierarchy, and clean organization. CSS will be added in future iterations to enhance layout and visual styling.

## Featured Project

### Eira — Personal Health Companion (In Progress)

Eira is an **AI-assisted health journaling web app** that converts natural, chat-style messages into structured entries. Users can identify patterns in their health through a simple and streamlined system.

**Tech stack:** Next.js | TypeScript | Tailwind CSS | PostgreSQL  
**Live demo:** [https://eira-phi.vercel.app](https://eira-phi.vercel.app)
